# Master Glossary

> **This is the sole authority for term renderings.** Before translating, read
> this file. Before using a technical term, check this file. If a term is not
> listed, add it here FIRST, then use it in translation.

## Conventions

- **Sanskrit form** is the canonical reference form (not Pali, not Tibetan).
- **English rendering** is used in running text.
- **First-use pattern:** "English rendering (Sanskrit)" on first occurrence in each fascicle. Thereafter, either English or Sanskrit per the "Preferred in text" column.
- **NTC register:** Commonly known Sanskrit terms are preserved in running text.
