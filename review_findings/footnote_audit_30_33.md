# Footnote Audit: Fascicles 30–33

**Auditor role:** Scholarly editor reviewing footnotes for a print translation aimed at 80% practitioners, 20% scholars. The test for every footnote: does it earn its place, or does it add weight without adding light?

**Fascicle counts:**
- Fascicle 30: 28 footnotes
- Fascicle 31: 27 footnotes
- Fascicle 32: 51 footnotes
- Fascicle 33: 38 footnotes
- **Total across four fascicles: 144 footnotes**

### Useful but Could Be Trimmed (6)

**[^three_dwelling_types]** — Useful, but overlong. The note explains the three meditation settings (forest, tree-base, empty room) with a canonical reference, then adds the Yogacara teacher-student point. The canonical sutta reference (MN 10, DN 22) and the fifth quality observation are worth keeping, but the middle description of each dwelling type restates what is already in the body text. **Cut roughly 40 words.**

**[^encouragement_similes]** — Useful, but the note lists all eight similes by category, then gives sutra parallels — the category breakdown duplicates what a reader just read. The sutra references (MN 56, MN 62) are the real value. **Trim to: note the eight-part rhetorical structure and the sutra parallels only.**

**[^four_preparations]** — Over-extended. The first two sentences explaining the four preparations are useful; the final sentence linking them to the "four reliances" (*catvari-pratisaranani*) is an interesting parallel but takes the reader well outside the passage. **Cut the four-reliances comparison; keep the Yogacara characterization.**

**[^exclusive_refuge]** — Useful but forward-referencing note is marginally useful. The forward reference to section 29.12 assumes linear reading and may confuse. **Shorten to the first sentence only.**

**[^three_vehicles]** — Useful but the Lotus Sutra contrast (*ekayana* vs. *triyana*) is a significant sectarian aside for a footnote. Core point (this is distinctively Yogacara) is worth stating; the Lotus counterpoint is a digression. **Trim the Lotus Sutra comparison.**

**[^disposition_signs]** — Useful but could be shorter. The note lists all five conduct-type physical indicators (which were just described in the body) and then says the Visuddhimagga has "a parallel but distinct set." The Visuddhimagga comparison is the only footnote-worthy content. **Cut the enumeration; keep only the Visuddhimagga reference.**

### Redundant (2)

The note explains "samadhi resources" (*samadhi-sambhara*) = moral discipline, sense restraint, moderation in food, etc. — but these are explicitly listed in the body text of this section, and the glossary has *samadhi-sambhara*. **[^three_object_types]** — Borderline redundant. The three object types were introduced in Fascicle 26 and the note correctly points back to sections 26.4-26.5. ### Excessive (3)

**[^four_vipashyana_distinction]** — Excessive. This is a second footnote on the same four vipashyana types, duplicating [^four_vipashyana] which appears just above it on the same passage. The [^four_vipashyana_distinction] note provides the Sanskrit reconstructions, then explains the *yavad-bhavikata*/*yathavad-bhavikata* distinction in more detail, then cites Abhidharmakosha II.33. For the target reader this is Abhidharma technical detail that goes beyond the passage. **Remove [^four_vipashyana_distinction] entirely; fold any essential point (the Sanskrit pair) into [^four_vipashyana].**

**[^equality_argument]** — Excessive in scope. The note identifies all eight arguments Asanga marshals for impartiality, then traces the "beginningless samsara" argument to Samyutta Nikaya, then links to Mahayana bodhicitta practice. The SN citation is valuable; the rest is analytical meta-commentary on the body text structure that a practitioner reader does not need spelled out. **Trim to: note the SN reference and indicate this is a standard maitri-extension argument.**

**[^nine_stages_parallels]** — Excessive. [^nine_stages] already covers the Tsongkhapa connection; this second note on the same passage adds Kamalashila, Mipham, and Visuddhimagga comparisons. The information is accurate but the reader has already gotten the essential context from [^nine_stages]. **Remove and fold the Kamalashila reference into [^nine_stages] if needed.**

### Candidate for Removal (1)

**[^three_understandings]** — Candidate for removal. The note provides the Chinese characters for the three understanding categories and says "these three exhaustively categorize the yogin's epistemic domain." The body text already explains all three understanding types just above this footnote in plain English. The Chinese-character identification is glossary/appendix material, not a footnote. This adds no understanding for the target reader. **Remove.**

It then distinguishes between form-sign apprehension (only body foundation) versus all-five-skandhas apprehension (all four foundations). **[^disputation_chain]** — Excessive detail. The interpretive commentary is not needed; the cosmological identification is minimal-value. **Remove.**

**[^fire_drill]** — Candidate for removal. This is a cross-reference footnote; its content (the simile appeared earlier; these terms were defined in 26.5) is not valuable enough to occupy a footnote. **[^two_levels]** — Candidate for removal. The note explains "vast and complete attainments, worldly and supramundane" = dhyanas + noble paths and fruits. This is a direct paraphrase of the body text phrase. The note adds nothing the body text does not already say more elegantly. **Remove.**

**[^cosmic_water]** — See "Useful but could be trimmed" above; if trimmed heavily, it becomes adequate. As currently written, however, the element-by-element mapping makes it a candidate for significant reduction.
