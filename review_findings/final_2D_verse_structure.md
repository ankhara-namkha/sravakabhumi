## 2. Verse Quality Assessment

### 2.1 F26.1 — Main Uddana [T30.0424a07]

**Chinese (8 lines, 4 couplets):**
```
諸補特伽羅　　建立所緣教
學隨順學法　　壞瑜伽作意
瑜伽師作修　　果門數取趣
因魔事無果　　是皆當廣說
```

**English rendering:**
> Persons, establishment, objects, instruction,
> Training, concordant dharmas, deterioration, yoga, attention,
> Yogins, activities, cultivation, fruits, synonyms, persons,
> Causes, mara-activities, fruitlessness — all shall be explained in full.

**Assessment:** ADEQUATE. The verse functions as a topic list (*uddana*), which by its nature is more catalog than poetry. The English captures the parallel structure. Line 2 is longer than the others (nine items vs. four each for lines 1, 3, 4), creating slight rhythmic imbalance, but this mirrors the Chinese compression. The footnote correctly explains the dual rendering of pudgala. **Rating: Acceptable (catalog verse).**

### 2.2 F26.5 — First Sutric Verse [T30.0428c02]

**Chinese:**
```
行者行諸相　　知一切實義
常於影靜慮　　得證遍清淨
```

**English rendering:**
> The practitioner practices the signs;
> knowing all true meaning,
> constantly upon the image in dhyana —
> they attain realization and complete purification.

**Assessment:** ADEQUATE. Four lines with a reasonable verse feel. The semicolon after line 1 and the em-dash after line 3 provide rhythmic pauses. "The image in dhyana" for 影靜慮 is slightly opaque but is explained in the prose commentary that follows immediately, which is the traditional structure (verse + commentary). **Rating: Acceptable.**

### 2.3 F26.5 — Second Sutric Verse [T30.0428c10]

**Chinese:**
```
於心相遍知　　能受遠離味
靜慮常委念　　受喜樂離染
```

**English rendering:**

**Assessment:** GOOD. Clean four-line verse. "The taste of seclusion" is a felicitous rendering of 遠離味. The participial structure creates natural parallel lines. **Rating: Good.**

**Summary of Q&A rendering conventions found:**

| Fascicle | Format Used | Chinese Pattern |
|----------|------------|-----------------|
| F26 | `**Question:**` / `**Answer:**` | 問/答 explicit dialogue |
| F27 | `Question:` / `Answer:` (unbolded) | 問/答 explicit dialogue |
| F28 | `**Question:**` / `**Answer:**` | 問/答 explicit dialogue |
| F29 | `**Question:**` / `**Answer:**` | 問/答 explicit dialogue |
| F30 | `**Question:**` / `**Answer:**` (where 問/答) | 問/答 explicit dialogue |
| F31 | Prose question embedded in paragraph | 云何X. 謂... (no explicit 問/答) |
| F32 | `**Question:**` / `**Answer:**` | 問/答 explicit dialogue |

This matches the Chinese source, which uses 云何X 謂... without explicit 問/答 dialogue markers in those passages — the Q&A is implicit rather than explicit. This differentiation is appropriate.

**Observation:** Fascicle 33 uses `*Question:*` / `*Answer:*` (italics) while most other fascicles use `**Question:**` / `**Answer:**` (bold). This is a minor inconsistency in formatting but does not affect comprehension.

## 6. Issues and Findings Summary

### Issue 1 — STRUCTURAL CONCERN (Moderate): F28 Non-Verse Blockquote

**Location:** F28_en.md, lines 149–169 (eleven drawbacks of yogic attention)

**What was found:** The eleven drawbacks (十一種) under drawback type (7) are formatted using blockquote syntax (`>`), even though the Chinese source presents them as a standard numbered prose list with no verse markers. The Chinese at T30.0436c11 reads: 謂十一種。一於諸諦寶蘊業果中猶預疑惑。二... This is a 謂 + enumeration structure — prose, not verse.

**Why this matters:** The blockquote convention in this translation signals verse passages. Using it for a prose enumeration creates a false visual equivalence between verse and list.

**Recommendation:** Reformat the eleven drawbacks as a standard numbered list (1–11) consistent with how other enumerations appear throughout the translation. The current blockquote (i)–(xi) format should be converted to:
```
1. Vacillating doubt regarding the truths...
2. Physical heaviness...
```
...and so on through 11.

### Issue 2 — FORMATTING INCONSISTENCY (Minor): F33 Q&A Italics vs. Bold

**Location:** F33_en.md throughout

**What was found:** F33 uses `*Question:*` / `*Answer:*` (italics) for Q&A markers. All other fascicles (F26, F27, F28, F29, F30, F32) use `**Question:**` / `**Answer:**` (bold).

**Recommendation:** Harmonize F33 to use bold `**Question:**` / `**Answer:**` format for consistency with the full translation.

### Issue 3 — VERSE QUALITY NOTE (Informational): F31 Intermediate Uddana Stanzas 1-2

**Location:** F31_en.md lines 299–312

**What was found:** The intermediate uddana's three stanzas differ in verse quality. Stanza 3 (purifying obstacles) reads with genuine verse rhythm. Stanzas 1-2 (six-aspect vipashyana and nine approaches) are dense catalog verse. This is largely unavoidable — the Chinese source is itself dense enumeration in these stanzas.

**Recommendation:** No change required. The current rendering is adequate given source constraints. For future revision consideration: stanza 2 (nine approaches) might benefit from restructuring to reduce the crowding of nine items into four lines.

### Issue 4 — NO ISSUE: Fascicles with No Verse (F27, F28, F29, F32)

Confirmed: the four fascicles with no blockquotes in the English (F27, F29, F32) correctly have no verse passages in the Chinese source. F28 has no verse blockquotes (the blockquote found in F28 is the non-verse list identified in Issue 1).
